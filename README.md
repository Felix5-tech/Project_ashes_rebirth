# Project_ashes_rebirth
Um app instalável tanto no pc quanto android que terá o objetivo de tornar o processo de organização financeira, de tarefas e lazer em algo prazeroso com ideias de "gamificação", recompensa e aprendizado com os erros, assim como em um jogo (com sistema de pontuação e nível)

O projeto está sendo criado em flutter para android e pc.
